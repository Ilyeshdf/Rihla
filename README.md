# Rihla (رحلة) 🇩🇿

Rihla is a premium, AI-powered travel companion designed for modern explorers in Algeria. Built with Flutter, it focuses on safety, immersive planning, and community-driven discovery.

## ✨ Features
- **Mediterranean Horizon Design**: A stunning dark-themed UI/UX inspired by Algerian landscapes.
- **Smart AI Planner**: Personalized itineraries generated via the Groq Llama-3 engine.
- **Safety Center**: One-tap SOS, verified emergency contacts, and live journey tracking.
- **Community Marketplace**: A specialized store classifying buyers as Individuals, Groups, or Companies.
- **Explorer Gamification**: Unlock achievements, earn XP, and climb the Saharan leaderboard.

## 🚀 Getting Started
1. Clone the repository.
2. Run `flutter pub get`.
3. Add your `groqApiKey` in `lib/config/constants.dart`.
4. Launch on macOS, iOS, or Android: `flutter run`.

## 🛡️ Safety & Reliability
All itineraries prioritize safe routes and verified locations, providing real-time safety scores for your peace of mind.
